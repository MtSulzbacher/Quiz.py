# README para o Quiz de Programação

## Descrição

Este script Python é um quiz interativo que testa seus conhecimentos básicos de programação. Abrangendo tópicos como a diferença entre hardware e software, linguagens específicas para manipulação de banco de dados, conceitos de back-end e front-end, operações básicas em Python e comandos SQL, este quiz é uma maneira divertida de testar e melhorar seu entendimento sobre programação.

## Como Começar

Para iniciar o quiz, simplesmente execute o script em um ambiente Python. O programa perguntará se você deseja iniciar o quiz. Responda com `S` para começar.

## Perguntas do Quiz

O quiz consiste em 5 perguntas de múltipla escolha sobre os seguintes tópicos:

1. Diferença entre hardware e software.
2. Linguagem de programação para manipulação de banco de dados.
3. Diferença entre back-end e front-end.
4. Resultado de uma operação básica em Python.
5. Comando SQL para filtrar dados.

Para responder a cada pergunta, digite a letra correspondente à sua resposta e pressione Enter.

## Pontuação

Cada resposta correta soma 1 ponto à sua pontuação total. Ao final do quiz, sua pontuação será exibida. Se você acertar 4 ou mais perguntas, receberá uma mensagem de parabéns.

## Encerrando o Quiz

Caso escolha não iniciar o quiz ao responder a primeira pergunta, o programa será encerrado e nenhuma pontuação será obtida.

---

### Notas

- Este quiz é uma ferramenta de aprendizado básico e é melhor utilizado como uma forma de revisão ou introdução a conceitos de programação.
- Não há limitação de tempo para responder às perguntas, então sinta-se à vontade para pesquisar e aprender mais sobre os tópicos abordados conforme necessário.
